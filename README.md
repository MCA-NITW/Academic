# MCA Academic Resources - NIT Warangal

Centralized repository of academic coursework, exam papers, study materials, and assignments for the Master of Computer Applications (MCA) program at NIT Warangal.

## Overview

A comprehensive collection of academic resources organized by semester, covering all core and elective courses in the MCA curriculum. Designed to help current and incoming students with exam preparation, revision, and understanding course expectations.

## How to Use

1. **Browse by semester** -- Navigate to the relevant `Sem-X/` directory
2. **Find your subject** -- Each subject has its own folder
3. **Access materials** -- Exam papers are in `Exam Papers/`, study materials in `Study Materials/`
4. **Batch-specific** -- Some content is organized by batch year (e.g., `Batch 21-24'`)

## Contributing

Contributions from MCA students are welcome. To add materials:

1. Fork the repository
2. Add your materials to the appropriate semester and subject folder
3. Follow the existing naming conventions
4. Create a Pull Request with a description of what you're adding

**Guidelines:**
- Organize by semester, then by subject
- Separate exam papers from study materials
- Include batch/year information where applicable
- Do not upload copyrighted textbook PDFs
